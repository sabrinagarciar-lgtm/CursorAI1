# Exercise 8 — Social feed

The runnable **Create React App** project is in the **`exercise8/`** subfolder.

For the feature purpose (social **feed** with post cards, user info, content, images, like/comment/share, **comment threads**, **infinite scroll** + **post composer**, **Tailwind** styling), **setup**, **project structure**, and a **screenshot**, see **[exercise8/README.md](exercise8/README.md)**.

Quick start:

```bash
cd exercise8
npm install --legacy-peer-deps
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
