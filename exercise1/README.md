# Exercise 1 — E-commerce Product Card

The runnable **Create React App** project is in the **`exercise1/`** subfolder.

For the feature purpose, setup steps (including `npm install --legacy-peer-deps`), project structure, and a **screenshot of the demo page**, see **[exercise1/README.md](exercise1/README.md)**.

Quick start from the nested app directory:

```bash
cd exercise1
npm install --legacy-peer-deps
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
