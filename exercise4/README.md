# Exercise 4 — Analytics dashboard

The runnable **Create React App** project is in the **`exercise4/`** subfolder.

For the feature purpose (KPI cards, chart placeholders, data table, filters and date range; Tailwind; professional layout; **dark mode**), **setup**, **project structure**, and a **screenshot**, see **[exercise4/README.md](exercise4/README.md)**.

Quick start:

```bash
cd exercise4
npm install --legacy-peer-deps
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
