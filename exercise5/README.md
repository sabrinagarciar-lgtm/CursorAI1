# Exercise 5 — Product search (Playwright E2E)

The runnable **Create React App** project is in the **`exercise5/`** subfolder.

For the feature purpose (product search, category and **price** filters, **sort**, **pagination**, **Playwright E2E**, **empty results**, error UI via query flag), **setup**, **project structure**, and a **screenshot**, see **[exercise5/README.md](exercise5/README.md)**.

Quick start:

```bash
cd exercise5
npm install --legacy-peer-deps
npm start
```

Then open [http://localhost:3000](http://localhost:3000). Run E2E (server on port 3010):

```bash
npm run test:e2e:install   # first time only
npm run test:e2e
```
