# Exercise 7 — Kanban board

The runnable **Create React App** project is in the **`exercise7/`** subfolder.

For the feature purpose (Todo / In Progress / Done columns, task cards with assignees, due dates, priorities, **drag-and-drop** via `@hello-pangea/dnd`, **TypeScript** + **Tailwind** with **dark mode**), **setup**, **project structure**, and a **screenshot**, see **[exercise7/README.md](exercise7/README.md)**.

Quick start:

```bash
cd exercise7
npm install --legacy-peer-deps
npm start
```

Then open [http://localhost:3000](http://localhost:3000). For Playwright:

```bash
npm run test:e2e:install   # first time
npm run test:e2e
```
