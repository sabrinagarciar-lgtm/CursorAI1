# Exercise 3 — Settings panel

The runnable **Create React App** project is in the **`exercise3/`** subfolder.

For the feature purpose (tabbed Profile / Notifications / Privacy / Appearance with forms, toggles, dropdowns, save actions; Tailwind and **dark mode**; responsive, accessible UI), **setup**, **project structure**, and a **screenshot**, see **[exercise3/README.md](exercise3/README.md)**.

Quick start:

```bash
cd exercise3
npm install --legacy-peer-deps
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
