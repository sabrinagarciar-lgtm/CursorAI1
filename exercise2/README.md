# Exercise 2 — Responsive navigation bar

The runnable **Create React App** project is in the **`exercise2/`** subfolder.

For the feature purpose (responsive navbar with logo, menu items, search, profile dropdown, mobile hamburger; TypeScript and Tailwind; sticky header with smooth animations), **setup**, **project structure**, and a **screenshot of the demo page**, see **[exercise2/README.md](exercise2/README.md)**.

Quick start:

```bash
cd exercise2
npm install --legacy-peer-deps
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
